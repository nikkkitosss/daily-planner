# Daily Planner Frontend

React + TypeScript frontend for the Daily Planner backend API.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create env file:

```bash
cp .env.example .env
```

3. Start dev server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

## Environment

- `VITE_API_BASE_URL` - backend API base URL (default: `http://localhost:4000`)
